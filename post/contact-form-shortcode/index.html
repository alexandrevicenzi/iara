Contact form shortcode

Iara Authors in Themes
Jun 1, 2020 · 1 min read

This theme has built in shortcodes.

Shortcodes are simple snippets inside your content files calling built-in or custom templates.

You can use a service like FormSpark with this shortcode.

Usage

1

{{< contact action="https://submit-form.com/12345" method="POST" recaptcha="XYZ1234" >}}

You must set action, method and recaptcha are optional.

Google Recaptcha will only work if the form service provides integration, FormSpark has this feature.

Using Google Recaptcha is a good way to avoid spam and bots.
